🔧 1. Install Python (Already done)

* Version: Python 3.10.x
* Downloaded from: (https://www.python.org/downloads/release/python-3100)

> Do NOT use Python 3.13 for `face_recognition` and `dlib` — not fully supported yet.




 🧪 2. Create & Activate Virtual Environment

Open PowerShell or VS Code terminal:

```bash
cd D:\SmartClassroomMonitoring
python -m venv venv
```

Then activate it:

```bash
.\venv\Scripts\activate
```

> If you get a script permission error, run this **once**:

```powershell
Set-ExecutionPolicy RemoteSigned
```

Confirm with `Y`, and try activate again.

---

## 📦 3. Install Required Libraries

Install these packages inside the virtual environment:

```bash
pip install flask opencv-python twilio schedule
```

---

⚠️ Face Recognition & dlib Setup (Manual due to build issues)

1. Install CMake:
   From (https://cmake.org/download) → Add to PATH during installation.

2. Install Visual Studio Build Tools
   From (https://visualstudio.microsoft.com/visual-cpp-build-tools) → Select Desktop development with C++

3. Download `dlib` .whl file (for Python 3.10)
   From: (https://www.lfd.uci.edu/~gohlke/pythonlibs/#dlib)

   File used: `dlib‑19.22.99‑cp310‑cp310‑win_amd64.whl`

   Install it:

   ```bash
   pip install path\to\dlib‑19.22.99‑cp310‑cp310‑win_amd64.whl
   ```

4. Install face\_recognition (skip dependencies):

   ```bash
   pip install face_recognition --no-deps
   ```

5. Manually install the rest of dependencies:

   ```bash
   pip install face-recognition-models numpy Pillow
   ```

---



4. Run Your Flask App

Every time you want to run your project:

```bash
cd D:\SmartClassroomMonitoring
.\venv\Scripts\activate
python app.py
```

Then open your browser and go to:

```
http://127.0.0.1:5000
```

To stop:
CTRL + C
